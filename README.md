# Sprint_6 - Автотесты для Яндекс.Самокат

## Описание проекта
Данный проект содержит автотесты для сервиса Яндекс.Самокат, разработанные с использованием Selenium WebDriver и pytest на языке Python. Тесты охватывают функциональность, связанную с заказом самоката и проверкой выпадающего списка в разделе «Вопросы о важном».

## Описание тестов
### Проверка выпадающего списка в разделе «Вопросы о важном»
- Проверка, что при нажатии на стрелочку открывается соответствующий текст. Для каждого вопроса написан отдельный тест.

### Заказ самоката
- Проверка всего флоу позитивного сценария с двумя наборами данных.
- Проверка точек входа в сценарий: кнопка «Заказать» вверху и внизу страницы.
- Проверка успешного создания заказа и появления всплывающего окна.

- Проверка перехода на главную страницу «Самоката» по клику на логотип.

- Проверка перехода на главную страницу Дзена по клику на логотип Яндекса.

## Установка и запуск
1. Клонируйте репозиторий:

   ```bash
   git clone <URL вашего репозитория>
   cd Sprint_6
   ```

2. Создайте и активируйте виртуальное окружение:

   ```bash
   python -m venv venv
   source venv/bin/activate  # Для Windows: venv\Scripts\activate
   ```

3. Установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```

4. Скачайте драйверы браузеров и поместите их в директорию `drivers/`:
   - ChromeDriver (Selenium 3): [Скачать](https://chromedriver.chromium.org/downloads)
   - GeckoDriver (для Firefox, Selenium 3): [Скачать](https://github.com/mozilla/geckodriver/releases)

5. Запустите тесты:

   ```bash
   pytest --alluredir=allure-results
   ```

## Используемые технологии
- Python
- Selenium WebDriver версии 4
- pytest

## Запуск Тестов
### Команда для Запуска
Для запуска всех тестов используйте следующую команду в терминале из корневой директории проекта:

```bash
pytest
```

## Генерация отчета Allure
Для генерации отчета по результатам тестирования выполните следующую команду:

```bash
pytest --alluredir=allure-results
```

Эта команда создаст временный сервер и откроет отчет в вашем браузере.
```bash
allure serve allure-results
```
